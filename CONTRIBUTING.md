# Contributing Guidelines

Thank you for your interest in contributing to the Shipping Calculator project. We welcome all contributions that help improve this project.

## How to Fork the Repository

1. Go to the GitHub repository page.
2. Click the "Fork" button in the top-right corner.
3. This will create a copy of the repository in your GitHub account.

## How to Create a Branch

1. Clone your forked repository:
   git clone https://github.com/your-username/shipping-calculator.git

2. Navigate to the project directory:
   cd shipping-calculator

3. Create a new branch for your changes:
   git checkout -b feature-branch-name

Use a meaningful branch name related to your work.

## Commit Message Guidelines

- Write clear and short commit messages.
- Use present tense (example: "Add shipping cost calculation").
- Describe what the commit does.
- Example commit message:
  Add validation for shipping weight

## How to Submit Pull Requests

1. Push your changes to your forked repository:
   git push origin feature-branch-name

2. Go to the original repository on GitHub.
3. Click "Compare & pull request".
4. Add a clear title and description of your changes.
5. Submit the pull request for review.

## Code of Conduct

By contributing to this project, you agree to follow the project's Code of Conduct and help maintain a respectful environment.
